SQL Cheat Sheet

SELECT usage

Selects all the rows from the table

SELECT * FROM tablename

Selects from certain fields from the table

SELECT field_name1, field_name2
FROM tablename

Selects from field in the table AND orders it by criteria

SELECT field_name
FROM tablename
ORDER BY category ASC (ASC or DESC)

JOIN usage

Joins rows from both tables.

SELECT table1.field, table2.field
FROM table1
INNER JOIN table
ON table1.id = table2.region_id
ORDER BY region_id ASC;

WHERE usage

Allows you to select the fieldname from the table in the selected (where) column (fieldname2).

SELECT fieldname
FROM table
WHERE fieldname = "fieldname2"
